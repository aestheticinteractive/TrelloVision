The Easy Way!

Visit the Overview module. Click one of your board links. Done.

The following tutorials describe how to view any public board or list via ID value.

Board Tutorial

Every Trello board has a long ID string at the end of its URL. The ID is 4d5ea62fd76aa1136000000c for this board:
https://trello.com/board/trello-development/4d5ea62fd76aa1136000000c

The base Card Table URL for a board is:
http://aestheticinteractive.github.io/TrelloVision/#/cardtable/board

Put these two peices together to visit the board's Card Table:
http://aestheticinteractive.github.io/TrelloVision/#/cardtable/board/4d5ea62fd76aa1136000000c

Please Note

Boards and lists can become very large. It sometimes takes several seconds for a TrelloVision page to load the necessary Trello data.
